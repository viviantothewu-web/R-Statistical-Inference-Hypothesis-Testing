# Statistical Inference & Hypothesis Testing in R

This project features advanced statistical workflows in R, focusing on probability distributions, inferential testing, and error analysis.

## 📊 Project Scope

### 1. Goodness-of-Fit & Independence Tests
**Claim Validation:** Used Likelihood Ratio test statistics ($G^2$) to validate manufacturer proportions (M&M's color distribution)
**Algorithm Testing:** Verified computer-generated Binomial distributions using 9 degrees of freedom tests
**Contingency Tables:** Analyzed Hodgkin's disease treatment responses via $4 \times 3$ contingency matrices to test for variable independence

### 3. Power & Error Analysis

* **Language:** R
